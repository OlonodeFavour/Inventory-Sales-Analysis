## 📝 Overview
This project presents an Inventory Management Dashboard built using Microsoft Excel.  
The dashboard provides a clear and interactive view of inventory performance, including stock levels, sales, profit, and cost across different store locations.

It is designed to support data-driven decision-making in inventory control and business performance tracking.

---

## ❗ Problem Statement
Businesses often face challenges such as:
- Lack of real-time visibility into inventory levels  
- Inefficient restocking decisions  
- Uneven sales performance across locations  
- Difficulty tracking profit and cost trends  

This project addresses these issues by creating a centralized dashboard that simplifies monitoring and improves decision-making.

---

## 🛠️ Tools Used
- **Microsoft Excel**

---

## ⚙️ Methodology
- Collected inventory dataset containing product, sales, cost, and location data  
- Structured and organized data for analysis  
- Created calculated metrics such as Total Sales, Profit, and Restock Quantity  
- Used Pivot Tables to summarize and analyze data  
- Built a dashboard using charts and KPI cards for visualization  

---

## 🧹 Data Cleaning and Preparation
- Removed duplicate records  
- Handled missing or inconsistent values  
- Standardized product names and categories  
- Ensured correct data types for numerical fields  
- Created new calculated columns:
  - Profit = Sales - Cost  
  - Restock Quantity  

---

## 📌 Key Insights / Findings
- 67% of inventory is still available, while 33% has been sold  
- Lagos recorded the highest sales among all locations  
- Some products have high availability but still require restocking  
- Profit trends fluctuate across months, indicating possible seasonal effects  
- Costs are relatively high compared to profit, suggesting inefficiencies  

---

## 💡 Recommendations
- Implement promotional strategies to reduce excess inventory  
- Focus on high-performing locations and replicate their strategies  
- Introduce automated alerts for low-stock items  
- Optimize operational costs to improve profit margins  
- Analyze seasonal trends to better plan inventory and sales
